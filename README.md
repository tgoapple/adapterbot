# adapterBot 🤖❤️📸

**Automate your customer service on Instagram.**

adapterBot creates an Instagram-style landing page with an AI chatbot embedded in it. Drop the link in your Instagram bio. Customers tap and chat — no app, no website, no friction.

### How it works

1. Train the bot on your business (hours, menu, FAQs, voice)
2. Deploy this page anywhere — Vercel, Netlify, Railway, your own server
3. Drop the link in your Instagram bio

### Free deployment

This is a single HTML file. No build step, no dependencies.

```bash
# Clone it
git clone https://github.com/tgoapple/adapterbot.git

# Open it locally
open index.html

# Or deploy on Vercel in 30 seconds
vercel --prod
```
